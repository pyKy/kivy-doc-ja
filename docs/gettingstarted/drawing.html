描画¶

各Widgetは、キャンバス(例:描画スペース)を持っています。キャンバスは、描画命令の集まりです。Widgetのグラフィカルな表現が再描画がされるたびに実行されるべきものです。

context 命令と vertex 命令：キャンバスに2種類の命令を追加することができます。変化した場合に自動的に更新されることです。 Pythonの場合、自分で行う必要があります。

どちらの場合も、MyWidget`の ``position` や size が変更されるたびに、キャンバスは再描画されます。

それらを実行したいときに基づき、あなたの指示で別個にするために canvas.before または canvas.after グループを使用することができます。Kivyグラフィックスの処理方法での詳細な外観については、こちらをご覧ください。

Kivyグラフィックスの処理方法での詳細な外観については、:mod:`こちら <kivy.graphics>`をご覧ください。
