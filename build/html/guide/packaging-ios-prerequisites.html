IOS Prerequisites(翻訳済み)¶

このガイドは以下を前提としています。

  • XCode 5.1以上

  • OS X 10.9以上

経験していたものとバージョンがと異なる場合があります。

Getting started(はじめに)¶

iTunesストアへ申請するためには、`iOS Developer License <https://developer.apple.com/programs/ios/>`_が必要になります。テストのために、物理デバイスまたはXCodeのiOSのエミュレータを使用できます。

デバイス上でテストするために、これらのデバイスを登録し、「プロビジョニングプロファイル」をインストールする必要がありますのでご注意ください。詳しくはAppleのApple の Getting started を参照してください。

Homebrew¶

Kivyで使用される依存関係やツールのいくつかをインストールするには、OSXの`Homebrew <http://brew.sh/>`を使用します。本当に便利なツールで、オープンソースプロジェクトで`Github <https://github.com/Homebrew/homebrew>`_にホストされています。

パッケージ管理（バージョンおよびオペレーティングシステムとの相性）の性質のために、このプロセスは、エラーが発生しやすくなり、ビルドプロセスに障害を引き起こします。メッセージは、通常、このようなエラーです。「The Missing requirement: <pkg> is not installed!」メッセージは、通常、このようなエラーです。

最初にこのコマンドを実行しているか確認すべきです:

brew install autoconf automake libtool pkg-config mercurial
brew link libtool
brew link mercurial
sudo easy_install pip
sudo pip install cython

それでもエラーが発生した場合はHomebrewが正常な状態にあるか確認してください。

brew doctor

更なるヘルプについては`Homebrew wiki <https://github.com/Homebrew/homebrew/wiki>`_ を参照してください。

最後は、最終的かつ絶望的させるための方法は、Homebrewを削除してから、最新バージョンを取得してインストールしなおすと、

`Mac の OSX 用のHomebrewをアンインストールして削除する方法<http://www.curvve.com/blog/guides/2013/uninstall-homebrew-mac-osx/>`_
