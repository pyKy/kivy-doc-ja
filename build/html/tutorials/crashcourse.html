Crash Course(翻訳済み)¶

Kivyクラッシュコースは、Kivyのコアデベロッパーによる一連のYouTubeビデオチュートリアルです。 Pythonでコーディングする方法を知っているPythonの初心者を対象とした、Kivyの簡単なウォークスルーを提供します。 PongとPaintのチュートリアルの後、このビデオセットは、コードをエレガントで見やすいものに保ちながら、アプリケーションをより迅速に作成するために使用できる基本的な機能とテクニックをカバーしています。

Basic Info(基本情報)¶

クラッシュコースは主に、それぞれ約10分程度の一連のYouTube動画で構成されています。また、ビデオに使用されているコードの一部を説明した記事もあります
